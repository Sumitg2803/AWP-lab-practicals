Student Management System

A comprehensive desktop application for managing student records digitally

Completed: February 2025
Python & MySQL
Desktop Application

Overview

A desktop-based application built with Python (Tkinter) and MySQL to manage student records digitally. It provides a simple GUI to add, update, delete, search, and view student data, ensuring accuracy and reducing manual work.

Tech Stack

Python

Tkinter for GUI, MySQL connector for database

MySQL

Data storage and management

Pillow (PIL)

Image processing and handling

TTK Widgets

Modern UI components

Key Features

Add New Records

Add new student records with full details including personal information, contact details, and academic data

Update & Delete

Easily update existing records or delete outdated entries with confirmation dialogs

Advanced Search

Search by Roll Number, Phone Number, or Student ID for quick data retrieval

Data Display

Display all data in a structured table format with scrollbars for easy navigation

User-Friendly Design

Intuitive interface with message popups and clear navigation for better user experience
